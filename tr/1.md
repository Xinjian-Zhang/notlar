### 1.3.元音和谐规则

将原因分为2组：前元音，后元音
- 后元音：Aa, Iı, Oo, Uu
- 前元音：Ee, İi, Öö, Üü

**1 复数**

- 前元音 + ler
- 后元音 + lar

| | 单数 | 复数 |
| ---- | ---- | ---- |
| 小孩(后元音) | çocuk | çocuklar |
| 人(后元音) | insan | insanlar |
| 礼物(前元音) | hediye | hediyeler |
| 花(前元音) | çiçek | çiçekler |

**2 疑问语气词**<br>

共有4个疑问语气词：mı, mi, mu, mü<br>

1.后元音
- 以a/ı结尾 + mı
  - Hasta mı? 生病了吗？
- 以o/u结尾 + mu
  - yok mu? 没有吗？

2.前元音
- 以e/i结尾 + mi
  - güzel mi? 漂亮吗？
- 以ö/ü结尾 + mü
  - yüz mü? 一百吗？
